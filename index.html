Why WumpBot?


What makes WumpBot so special? Why shouldn't I just use a popular bot like MEE6?

Those are the questions every server owner or admin should ask before inviting a bot. And I am here to answer these questions.

1. I care about user input. I want the bot to be useful for every server, no matter how big or small. And the best way to do that, is to listen to your(The users) suggestions. Leave a suggestion

2. WumpBot is an all-in-one bot. The bot has Moderation, fun, info, giveaways, level system and an economy system(Economy system currently not available due to errors.). This means, that there is no need to invite other bots like Auttaja, Arcane.

3. WumpBot is completely free. All the commands made for WumpBot are completely free. This is actually the main reason I made the bot. I hate premium only commands, and I am pretty sure that you do aswell.
